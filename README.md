AgentD



ABOUT SOFTWARE

* This is a document analyser software that uses generative AI to summarize and find insights on PDF documents.
* This software uses Gemini API via LangChain.



SETUP THE PROJECT

* Download this repository into your local machine.
* Go to this link "https://aistudio.google.com/" and get an 'API key' for using Gemini API.
* Paste the API key in to the "placeholder" field in the .env file.



RUN THE SOFTWARE

* open 'backend' folder and run 'server.py'.
* open 'frontendUI' folder in terminal and execute command: 'npx serve'.
* open 'https://localhost:3000' in any browser. (This will open the GUI of the software)
